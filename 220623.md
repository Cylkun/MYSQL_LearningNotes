### 视图数据的更新

> 5.5 版本可以更新视图, 同时会更新原始表, 8.0.28 版本不能更新

不允许更新的视图

1. 包含以下关键词: GROUP BY, DISTINCT, HAVING, UNION, UNION ALL, 分组函数
2. 常量视图
3. `SELECT` 中包含子查询
4. 视图是由一个不能更新的视图为基础创建的
5. `WHERE` 中的子查询使用了和 `FROM` 后的表相同的表

常量视图: `SELECT` 后面是一个常量


### 视图 vs 表

|                      | 表             | 视图            |
| -------------------- | -------------- | --------------- |
| 创建语法关键字       | `CREATE TABLE` | `CREATE VIEW`   |
| 是否实际占用物理空间 | 占用           | 只保存 SQL 逻辑 |
| 使用                 | 增删改查       | 一般不能增删改  |
